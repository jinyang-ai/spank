FAQ

Common questions, answered.

"Apple cannot verify" or app is "damaged"? ▼
This app isn't signed with Apple Developer ID ($99/year - not worth it for a free meme app). Run this in Terminal: xattr -cr /Applications/Spank.app
Nothing happens when I slap? ▼
1. Slap harder - threshold is set high to avoid false triggers
2. Check the 🍑 in menu bar - make sure it shows "Running"
3. Lower the threshold in the menu if needed
4. Try slapping the palm rest or side, not the screen!
Does it work on Intel Macs? ▼
No. Only Apple Silicon Macs (M1, M2, M3, M4) have the right accelerometer. Sorry Intel gang.
Will it void my warranty? ▼
No. The app just reads the accelerometer that Apple put in your Mac. Slapping is fine - unless you crack the screen.
How do I uninstall? ▼
Run this in Terminal: sudo launchctl unload /Library/LaunchDaemons/com.aj.spank.plist && sudo rm /Library/LaunchDaemons/com.aj.spank.plist && sudo rm /usr/local/bin/spank && rm -rf /Applications/Spank.app && rm -rf /Users/Shared/spank-sounds
