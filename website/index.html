INSTALLATION

Takes 60 seconds. One-time setup.

Why extra steps? This app isn't signed with Apple Developer ID ($99/year — not worth it for a free meme app). The commands below just remove macOS's quarantine flag.
01

DOWNLOAD & MOUNT

Double-click Spank.dmg to open it

02

DRAG TO APPLICATIONS

Drag Spank.app into your Applications folder

03

OPEN TERMINAL

Press ⌘ + Space, type Terminal, hit Enter

04

REMOVE GATEKEEPER BLOCK

Copy-paste this command and press Enter:

xattr -cr /Applications/Spank.app
Click to copy
05

RUN INSTALLER

Copy-paste this command (enter your password when asked):

sudo /Applications/Spank.app/Contents/Resources/install-simple.sh
Click to copy — Installs background service (one-time)
06

LAUNCH THE APP

Open Spank.app from Applications. Look for 🍑 in your menu bar.

07

SLAP YOUR MACBOOK

Give it a hard slap on the side or palm rest. Trust us.

FAQ

Common questions, answered.

"Apple cannot verify" or app is "damaged"? ▼
This app isn't signed with Apple ($99/year — pass). macOS blocks unsigned apps by default.

Fix: Open Terminal and run: xattr -cr /Applications/Spank.app This removes the quarantine flag. Safe — check the source on GitHub if paranoid.
Will it void my warranty? ▼
No. The app just reads the accelerometer Apple put in your Mac. Slapping is fine — unless you crack the screen.
Does it work on Intel Macs? ▼
No. Only Apple Silicon Macs (M1 Pro, M2, M3, M4) have the right accelerometer. Sorry Intel gang.
Nothing happens when I slap? ▼
1. Slap harder — threshold is set high to avoid false triggers
2. Check the 🍑 in menu bar — make sure it shows "Running"
3. Lower the threshold slider if needed
Does it drain battery? ▼
Barely. CPU usage is near zero — it just listens to the accelerometer.
How do I uninstall? ▼
Run this in Terminal: sudo launchctl unload /Library/LaunchDaemons/com.aj.spank.plist && sudo rm /Library/LaunchDaemons/com.aj.spank.plist && rm -rf /Applications/Spank.app
